# H5 作品名片

这是面向国内移动端用户的 H5 发布版。用户可以通过链接或二维码在微信内置浏览器、手机浏览器中直接访问。

## 文件

- `index.html`: 单文件静态页面，包含编辑器、作品名片预览、草稿保存、带数据的分享链接和二维码展示。

## 部署建议

建议部署到国内静态资源服务：

- 腾讯云 COS + CDN
- 阿里云 OSS + CDN
- 其他支持 HTTPS 的国内静态托管

如果部署到国内服务器或 CDN，需要按服务商要求完成域名备案。

## 注意

- 生成名片后，页面会把名片数据编码到链接里；用户通过链接或二维码打开时，可以直接看到同一张名片。
- 二维码当前使用公开二维码服务生成，正式商业化建议换成自有后端或部署本地二维码库，避免国内访问不稳定。
- 当前页面不依赖海外字体或海外 JS CDN。
- 当前草稿保存在浏览器本地，没有账号和云端同步。
