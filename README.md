# 作品名片

把零散经历整理成可信作品名片，用于求职、合作和接单。

## 目录

- `index.html`: Web 预览版
- `h5/`: 面向国内手机用户的 H5 发布版，一个链接或二维码即可访问
- `miniprogram/`: 微信小程序版

## 小程序预览

使用微信开发者工具导入本仓库根目录。当前 `project.config.json` 使用 `touristappid`，正式上线前需要替换为你的真实小程序 AppID。

## 上线前注意

- 小程序无法直接跳转任意外部链接，作品链接当前采用“复制链接”的方式。
- 当前版本使用本地存储保存草稿，没有账号体系和云端同步。
- 正式发布前需要补充小程序隐私协议、类目、服务内容说明和真实 AppID。

## H5 部署

H5 发布版在 `h5/index.html`。建议部署到国内静态托管和 CDN，并使用 HTTPS 域名生成二维码分享。
