# 作品名片

把零散经历整理成可信作品名片，用于求职、合作和接单。

## 目录

- `index.html`: Web 预览版
- `h5/`: 面向国内手机用户的 H5 发布版，支持发布短链接和二维码访问
- `server/`: H5 后端服务，保存作品名片并生成 `/c/短ID` 访问链接
- `miniprogram/`: 微信小程序版

## 小程序预览

使用微信开发者工具导入本仓库根目录。当前 `project.config.json` 已配置 AppID `wx5331540c7004bfcd`。

## H5 本地预览

```bash
npm start
```

启动后访问 `http://localhost:8787/h5/index.html`。填写信息并点击“发布并生成链接”后，会生成 `/c/短ID` 的作品名片链接，别人打开后只看到名片展示页。

## 上线前注意

- 小程序无法直接跳转任意外部链接，作品链接当前采用“复制链接”的方式。
- 当前版本使用本地存储保存草稿，没有账号体系和云端同步。
- 正式发布前需要补充小程序隐私协议、类目、服务内容说明和真实 AppID。

## H5 部署

第二阶段 H5 需要后端保存数据，不能只把 `h5/index.html` 扔到静态托管。建议部署到国内云服务器、腾讯云 CloudBase 或可运行 Node.js 的服务，并为域名配置 HTTPS。
